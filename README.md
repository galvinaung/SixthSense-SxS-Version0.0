# SixthSense (SxS)

## Smart Cross-Platform Device Security & Awareness System

Version: 0.0 Proof of Concept (PoC)

---

## Overview

SixthSense (SxS) is a personal device awareness and security monitoring system.

The goal of SixthSense is to notify the device owner when registered devices are used and provide activity history.

Version 0.0 focuses on creating a simple Windows desktop application that can:

- Identify the current Windows device
- Detect the current user
- Record local activity history
- Display previous activity records

---

## Version 0.0 Scope

Platform:

- Windows 11

## Features

### Device Information

Display:

- Computer Name
- Windows Username
- Operating System
- Date
- Time


### Local Activity History

Store:

- Date
- Time
- Computer Name
- Username


### History Viewer

View previous activity records stored locally.
